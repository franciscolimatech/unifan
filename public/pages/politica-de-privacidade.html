Política de Privacidade

Cristóvão - Plataforma de Gestão Omnichannel

Esta política é acadêmica e simulada. O projeto não possui backend, API, banco externo ou servidor próprio para autenticação.

1. Dados armazenados

Para simular o acesso à plataforma, o navegador pode armazenar nome, e-mail, hash de senha, pergunta de recuperação, hash da resposta e informações de aceite dos termos.

2. Onde os dados ficam

Os dados são salvos no localStorage do próprio navegador, usando chaves locais do protótipo. Eles não são enviados a um servidor do projeto e podem ser apagados limpando os dados do site no navegador.

3. Finalidade

  • Permitir login e cadastro simulados.
  • Registrar que o usuário aceitou os Termos de Uso e esta Política de Privacidade.
  • Demonstrar boas práticas de interface e consentimento em um protótipo acadêmico.

4. Segurança

Senhas e respostas de recuperação são salvas em formato de hash SHA-256 quando o navegador oferece esse recurso. Ainda assim, por ser um protótipo local, não utilize senhas reais ou dados sensíveis.

5. Controle do usuário

O usuário pode remover os dados limpando o armazenamento local do navegador. Como não há servidor real, o projeto não mantém cópias remotas desses dados.
